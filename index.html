Minha primeira aula de Git

O Git nos permite armazenar as modificações feitas em cada arquivo em um servidor próprio para isso. Toda a gestão de alterações é feita através do Git, dessa forma, a nossa preocupação deve ser em criar código que funciona, e não em quem alterou o que antes.

O commit é a forma de salvar um estado ou versão do nosso código; O parâmetro -m serve para passarmos uma mensagem de commit, essa mensagem deve está entre aspas.
